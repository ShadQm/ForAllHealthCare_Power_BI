# ForAll Health Care Dashboard
## Overview

The ForAll Health Care Dashboard is an interactive data visualization tool designed to monitor and analyze hospital patient information efficiently. It provides insights into patient admissions, discharges, billing amounts, and doctor assignments — enabling healthcare administrators and analysts to make informed decisions quickly.

This project centralizes patient and financial data into a clean, dynamic dashboard for easy access and reporting.

## Key Features

### Patient Financial Summary:
Displays the total amount per patient ID, including billed, paid, and outstanding balances.

### Admission & Discharge Tracking:
Visualizes each patient’s admission and discharge details, helping identify patient flow and occupancy trends.

### Doctor Assignment Overview:
Shows which doctor treated each patient, including the number of cases handled per doctor.

### Payment Analysis:
Highlights amount paid and pending payments to monitor financial performance and patient billing efficiency.

### Interactive Dashboard:
Built for ease of use with filters and visual charts to quickly access insights by patient, doctor, or date range.

### Dashboard Insights

Patient Summary Table: Displays ID, name, admission date, discharge date, attending doctor, and billing details.

Billing Graphs: Charts show the total billed amount per patient and per doctor.

Doctor Statistics: Displays total patients treated and revenue generated by each doctor.

Date Filters: Allows analysis for specific time periods (e.g., monthly, quarterly, or yearly).
